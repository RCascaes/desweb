Sobre este projeto

Estudei HTML e CSS há muitos anos, algo entorno de 11 anos atrás, em meu curso técnico em informática. Hoje cursando o ensino superior em análise e desenvolvimento de sistemas acabamos por aprender pouco de desenvolvimento web, onde no pouco, utilizamos PHP.

Busquei conhecimentos externos referente a HTML5, CSS3, JS e GIT, e este projeto é para fixar o aprendizado e praticar um pouco. Nele estarei buscando aperfeiçoar também a codificação, com boas práticas e melhor aproveitamento de código.

Foi utilizado o Trello para organização das etapas de desenvolvimento, o projeto foi dividido em partes, onde cada parte era um card, defini prioridades e estipulei data limite para a entrega de cada card.

Aviso:

Como o objetivo é visar o aprendizado utilizando apenas HTML, CSS e JS, esta página não funcionará, uma vez que esta área requer um servidor SMTP e configurações feitas no back-end.
